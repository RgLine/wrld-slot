🎰 WRLD Bot

Bot Ekonomi & Permainan Serbaguna untuk Server Discord

+ Tambahkan ke Server 🛟 Server Dukungan

Awalan bawaan saat ini: wrld

Pengelola server bisa mengubahnya sesuka hati, contoh:
wrld setprefix ! atau wrld setprefix bot
Maksimal 10 karakter.

ℹ️ Bantuan

wrld help — Tampilkan daftar semua perintah bot

💰 Perintah Ekonomi & Saldo

wrld balance / wrld cash — Cek saldo WRLD Balance
wrld daily — Ambil hadiah harian
wrld give @pengguna <jumlah> — Kirim saldo ke pengguna lain

🎮 Permainan & Taruhan

wrld slot / wrld s / wrlds <jumlah/all> — Main mesin slot
wrld cf <jumlah/all> [h/t] — Lempar koin, tebak kepala/ekor
wrld crate — Buka kotak senjata yang dimiliki

📊 Profil & Tingkat

wrld profile [@pengguna] — Lihat profil lengkap
wrld level [@pengguna] — Lihat tingkat, XP, dan peringkat

⚙️ Pengaturan Server (Khusus Admin)

wrld setprefix <teks_baru> — Ubah awalan perintah
wrld blockchannel — Matikan perintah di saluran ini
wrld allowchannel — Nyalakan kembali perintah
Semua data saldo, tingkat, dan pengaturan tersimpan aman.
Bot juga mendukung perintah garis miring /
